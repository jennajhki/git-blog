조건문과 반복문 (if, else, for, while)

📅 2025년 5월 13일

오늘은 JavaScript의 조건문과 반복문을 배웠다.

문법은 단순해 보이지만, 로직이 어떻게 되는지 구현하는 게 어렵고, 중첩 조건문이나 반복문 안에 조건문이 들어가는 경우 더 헷갈림.

1. 조건문 if / else

조건에 따라 코드를 다르게 실행할 때 사용하는 구문.


const score = 85;

if (score >= 90) {
  console.log("A");
} else if (score >= 80) {
  console.log("B");
} else {
  console.log("C or below");
}

2. 반복문 for

정해진 횟수만큼 반복할 때 사용.


for (let i = 1; 조건식; 증감식) {
  console.log(i);
}

for (let i = 0; i < 5; i++) {
  console.log("i의 값:", i);
}

3. 반복문 while

조건이 참인 동안 계속 반복. 조건이 false가 될 때까지 반복하므로, 종료 조건을 정확히 써야 무한루프를 피할 수 있음.


let count = 0;

while (count < 3) {
  console.log("count:", count);
  count++;
}

아직 헷갈려서 중첩 반복문도 연습 더 해봐야겠다..!
