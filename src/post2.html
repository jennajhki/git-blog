JavaScript 기초: 타입과 함수

📅 2025년 5월 12일

1. JavaScript의 기본 타입

오늘은 JavaScript의 기본적인 데이터 타입에 대해 배웠다. 자주 쓰이는 타입은 아래와 같다:

  • number – 숫자 (예: 10, 3.14)
  • string – 문자열 (예: "hello")
  • boolean – true 또는 false
  • null – 값이 없음을 명시적으로 표현
  • undefined – 값이 할당되지 않은 상태
  • symbol – 유일한 값을 나타내는 타입

JavaScript에서는 정수(int)와 소수(float)를 구분하지 않고 모두 number 타입으로 처리한다.


const a = 10;
const b = 3.14;

console.log(typeof a); // "number"
console.log(typeof b); // "number"

내부적으로는 IEEE-754 방식의 64비트 부동소수점으로 저장된다.

2. 아주 큰 정수: bigint

number 타입은 최대 안전 정수인 253 - 1까지만 정확하게 표현 가능하다. 이를 초과하는 경우 bigint 타입을 사용해야 한다.


const big = 999999999999999999999n;
console.log(typeof big); // "bigint"

숫자 뒤에 n을 붙이면 bigint로 인식된다.

3. 특수 숫자: NaN과 Infinity

자바스크립트에는 숫자와 관련된 특수 값도 있다:

  • NaN – Not a Number (숫자가 아님)
  • Infinity – 무한대

console.log(typeof NaN);      // "number"
console.log(typeof Infinity); // "number"

4. 정리 표

숫자 관련 타입들을 표로 정리하면 다음과 같다:

개념 JS 타입 예시 비고
정수 number 10, -3 정수와 소수 구분 없음
소수 number 3.14, 0.1 float 따로 없음
큰 정수 bigint 123456789123456789n 숫자 뒤에 n
무한대 number Infinity 예: 1 / 0
NaN number NaN Not a Number

5. typeof vs instanceof

typeof은 데이터 타입을 문자열로 확인하고, instanceof는 어떤 객체가 특정 생성자의 인스턴스인지 확인하는 데 사용된다.


const date = new Date();
console.log(typeof date);       // "object"
console.log(date instanceof Date); // true

2. 함수 작성 방법

JavaScript에서는 function 키워드를 사용해서 함수를 정의한다. 함수 이름 뒤에 () 소괄호를 쓰고, 그 안에는 매개변수를 넣을 수 있다. {} 중괄호 안에는 실행할 코드가 들어간다.

아래는 두 개의 숫자를 곱해주는 함수 예시다:


function multiply(x, y) {
  return x * y;
}

console.log(multiply(2934, 54392)); // 결과 출력

또는 아래와 같이 다른 함수를 정의하고 호출할 수도 있다:


function add(a, b) {
  return a + b;
}

let result = add(2, 3);
console.log(result); // 5 출력
