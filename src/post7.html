고차 함수 (filter, map, reduce)

📅 2025년 5월 20일

오늘은 JavaScript에서 고차 함수(Higher Order Function), 특히 filter, map, reduce에 대해 공부했다. for문이나 if문을 쓰던 방식과 다르게, 좀 더 추상적이고 선언적인 코드 방식이라는 점이 흥미로웠지만 동시에 아직 익숙하지 않아서 낯설게 느껴지기도 했다.

1. 고차 함수란(Higher Order Function)?

함수를 인자로 받거나 함수를 리턴하는 함수이다. JavaScript에서는 filter, map, reduce 같은 메서드가 대표적인 고차 함수이다.

2. 각 함수 설명 & 예시

📌 filter – 조건에 맞는 값만 골라내기


const nums = [1, 2, 3, 4, 5];
const even = nums.filter(n => n % 2 === 0); // [2, 4]

배열 중에서 조건에 맞는 값만 추출할 때 사용.

📌 map – 배열의 각 요소를 가공해서 새로운 배열 생성


const nums = [1, 2, 3];
const squared = nums.map(n => n * n); // [1, 4, 9]

배열을 변형하거나 계산할 때 사용.

📌 reduce – 배열을 하나의 값으로 누적


const nums = [1, 2, 3, 4];
const sum = nums.reduce((acc, cur) => acc + cur, 0); // 10

합계, 평균, 누적 결과를 만들 때 사용.

3. for문과 비교했을 때

기존에는 다음처럼 썼던 코드를:


// for문 방식
let result = [];
for (let i = 0; i < nums.length; i++) {
  if (nums[i] % 2 === 0) result.push(nums[i]);
}

이제는 이렇게 더 간결하고 선언적으로 작성할 수 있다:


// 고차 함수 방식
const result = nums.filter(n => n % 2 === 0);

4. 실습 예제: 배열 평균 구하기 (reduce 활용)

Number 타입을 요소로 갖는 배열을 입력받아 배열의 모든 요소의 평균을 리턴

조건은 반복문 사용 금지, reduce로 구현, 빈 배열 0 리턴


      function computeAverageOfNumbers(arr) {
        if (arr.length === 0) return 0;
      
        const sum = arr.reduce(function (acc, cur) {
          return acc + cur;
        }, 0);
      
        return sum / arr.length;
      }
      
      // 예시 출력
      console.log(computeAverageOfNumbers([1, 2, 3])); // 2
      console.log(computeAverageOfNumbers([]));       // 0

✔️ 해설

  • reduce를 사용해 acc에 누적합을 계산
  • 마지막에 배열 길이로 나눠 평균 계산
  • 빈 배열의 경우 length === 0 체크로 예외 처리

지금까지는 for문과 if문 중심의 사고방식에 익숙했기 때문에, 함수를 전달해서 처리한다는 개념이 아직은 어렵다.

하지만 코드를 간결하게 만들고, 반복적인 로직을 추상화하는 데 유용하다. map, filter, reduce를 더 많이 써보고 손에 익히는 연습이 필요하다.
