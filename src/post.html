오늘 배운 것: HTML, CSS활용 반응형 포트폴리오와 블로그 구조

📅 2025년 5월 9일

1. 포트폴리오 레이아웃 구조

어떤 컨셉으로 만들까 고민하다가 Spotify UI 스타일을 좋아해서 그 느낌으로 블로그 웹사이트의 기본 레이아웃을 만들었다. 어두운 배경에 로고와 메뉴를 상단에 배치했다.

2. 다크모드 스타일링

CSS에서 배경을 #121212, 텍스트를 #FFFFFF 등으로 지정해 전체적으로 세련된 다크 테마를 적용했다. 포인트 색상으로는 Spotify의 시그니처 컬러인 #1DB954를 사용해 시각적 통일성을 유지했다.

3. 블로그 페이지 만들기

개별 블로그 카드 형태의 레이아웃을 Flex/Grid를 활용해 구성하고, 추후 학습 기록을 카드로 추가할 수 있도록 구조화했다. 각각의 카드에는 제목, 날짜, 간단한 요약이 들어간다.

4. 앞으로의 방향

이 블로그는 앞으로 웹 개발을 공부하면서 배운 내용을 정리하는 공간으로 운영할 예정이다. 구조와 디자인 모두 스스로 다듬어가며 성장 기록을 쌓아나갈 계획이다.
