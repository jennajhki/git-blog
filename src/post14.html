Getting Started Python

📅 2025년 5월 30일

오늘부터 Python을 시작했다. JavaScript와 비슷한 점도 있지만, 다른 점도 많아서 새롭다.

파이썬 vs 자바스크립트

공부하면서 느낀 자바스크립트와 파이썬의 주요 차이점들

  • 문자열 여러 줄: ''' 또는 """ 사용 (JS는 \n 또는 백틱)
  • 비교 연산자: == (JS에서는 === 사용)
  • 변수 선언 키워드 없음 (let, const 사용 안 함)
  • 변수명: snake_case 사용

개인적으로 Python이 더 직관적이고 쉬운 것 같다.

Number Type

  • int: 100, -100
  • float: 1.23, -2.14
  • complex: 2+3j, -3-4j
Z = complex(2,3)
print(Z.real)  # 2.0
print(Z.imag)  # 3.0

복소수 계산

import cmath
z = 1 + 2j
print(cmath.polar(z))
print(cmath.phase(z))

수학 연산

import math
print(math.floor(100.621))  # 내림
print(math.ceil(100.621))   # 올림
print(math.round(100.621))  # 반올림
print(math.fabs(-100))      # 절대값
print(math.sqrt(4))         # 제곱근
print(math.pow(2, 5))       # 거듭제곱

String

여러 줄 문자열 예:

'''
Python은 배우기 쉬운 문법과 강력한 기능을 제공합니다.
다양한 분야에서 활용됩니다.
'''

문자열 연결

print("감사합니다." + " " + "🙏")
print(str(1) + "1")

문자열 인덱싱

str_var = "python"
print(str_var[0])
print(str_var[-1])

문자열 메서드

  • lower(), upper(): 대소문자 변환
  • join(), find(), count(): 연결, 검색, 개수 확인

Boolean

Falsy 값

  • False, 0, 0.0, "", [], {}, None

Truthy 값

  • True, 1, "text", [1], {"a": 1}

비교 연산자

print(123 == 100 + 23)  # True
print(123 != "123")  # True
print(100 > 200)  # False

논리 연산자

  • or: 논리합
  • and: 논리곱
  • not: 논리부정

변수

변수는 num = 2와 같이 선언하며, 재할당도 가능하다.

name = "Alice"
name = "Bob"  # 재할당 가능

상수

from typing import Final
PI: Final = 3.141592

변수명 규칙

  • 숫자로 시작 불가
  • 예약어 사용 불가

네이밍 컨벤션

  • snake_case: Python 권장 방식
  • camelCase: JavaScript 스타일
  • PascalCase: 클래스명에 사용
